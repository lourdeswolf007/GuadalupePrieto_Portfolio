Projects

Project One

My first project involved analyzing a dataset from Kaggle. This project focused on the Penguins Dataset on Kaggle, which includes information on penguin species, island habitats, and physical measurements.
It provides valuable insights into ecological trends.

For this project, I used the following tools:

Pandas, Tableau, Microsoft Excel, and GitHub Pages for deployment.

Project Two

My second project focused on Exploratory Data Analysis (EDA) to determine the correlation between the dry season and wildfires in California. To achieve this, I used the following statistical models:

Linear Regression, Descriptive Statistics, and Standard Deviation.

Project Three

My third project analyzed a week of ADA MBTA rides in Massachusetts through Exploratory Data Analysis. The following techniques were applied:

Data Cleaning using Excel formulas, Descriptive Statistics, and Histogram visualization.
